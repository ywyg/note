- [IP](./ip.md)

唯一的互联网地址：共有`A、B、C、D、E`五类互联网地址

DNS：分布式数据库，提供IP和域名的映射关系

TCP/IP采用CSMA/CD的媒体接入方法，速率为10Mb/s，目的地址和源地址为48bit

标准集：

- 802.3：针对CSMA/CD网络
- 802.4：针对令牌总线网络
- 802.5：针对令牌环网络

这三者的共性由 `802.2`定义





由于`TCP`由多种分层方式，我们只取一种分层方式分析，其他分层大同小异。

| 应用层 | FTP、E-Mail          |
| ------ | -------------------- |
| 运输层 | TCP、UDP             |
| 网络层 | IP、ICMP、IGMP       |
| 链路层 | 设备驱动程序与接口卡 |
