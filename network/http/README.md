HTTP是一种基于服务端和客户端的通讯解析；HTTP工作原理是通过发送请求和接受响应的方式来实现的；

URL语法：

```sh
<schema>://<user>:<password>@<host>:<port>/<path>;<params>?<query>#<frag>
```
