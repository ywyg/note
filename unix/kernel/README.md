## 内核（kernel）

### 什么是OS内核

> 常驻内存程序集
>
> 包含常用设备驱动程序、高频运行模块

### 目的是什么

1. 防止遭受其他应用程序破坏
2. 提高OS运行效率

### 功能

- 支撑功能
  - 中断处理
  - 时钟管理
  - 原语操作
- 资源管理功能
  - 进程管理
  - 存储器管理
  - 设备管理

## 处理机运行状态

- 内核态（系统态｜管态）

  > 能够执行一切指令，访问所有寄存器和存储区，OS处于这个层级

- 用户态（目态）

  > 仅能执行规定的指令，访问指定的寄存器
  >
  > 一般情况下，用户应用程序仅能在用户态运行
